## Getting started
1) Install Python 3.10+.
2) Optionally create a virtual environment: `python -m venv .venv && source .venv/bin/activate` (use `.venv\Scripts\activate` on Windows Command Prompt/PowerShell).
3) Add any required packages per exercise with `pip install -r requirements.txt` (if such a file is added later).

## How to use this repo
- Work through practical questions in `Prcticals-Q/`, keeping your solutions organized by topic or date.
- Try small experiments to reinforce concepts (control flow, functions, OOP, file I/O, modules, testing).
- Commit regularly to track progress and learn from iterations.
